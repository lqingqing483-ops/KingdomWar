# 主页面UI设计计划

## 任务目标
创建完整的主页面UI设计，包括所有必要的预制体、脚本和资源。

## 详细步骤

### 1. 创建UI目录结构
- 创建 `Assets/Prefabs/UIPrefab/` 目录
- 创建 `Assets/Prefabs/UIPrefab/Canvas.prefab` - UI画布
- 创建 `Assets/Prefabs/UIPrefab/Tip.prefab` - 提示信息
- 创建 `Assets/Prefabs/UIPrefab/promptMessage.prefab` - 弹窗消息

### 2. 创建主页面预制体
- 创建 `Assets/Prefabs/UIPrefab/mainPanel.prefab`
- 设计主页面布局，包括：
  - 顶部信息栏（等级、金币、钻石）
  - 左侧导航菜单（对战、卡牌、卡组、商店、任务、背包、设置）
  - 右侧主内容区
  - 底部状态栏

### 3. 创建相关面板预制体
- 创建 `Assets/Prefabs/UIPrefab/bagPanel.prefab` - 背包面板
- 创建 `Assets/Prefabs/UIPrefab/taskPanel.prefab` - 任务面板
- 创建 `Assets/Prefabs/UIPrefab/shopPanel.prefab` - 商店面板
- 创建 `Assets/Prefabs/UIPrefab/settingPanel.prefab` - 设置面板
- 创建 `Assets/Prefabs/UIPrefab/GetGoodsPanel.prefab` - 获取物品面板

### 4. 创建面板脚本
- 创建 `Assets/Scripts/UI/mainPanel.cs` - 主面板脚本
- 创建 `Assets/Scripts/UI/bagPanel.cs` - 背包面板脚本
- 创建 `Assets/Scripts/UI/taskPanel.cs` - 任务面板脚本
- 创建 `Assets/Scripts/UI/shopPanel.cs` - 商店面板脚本
- 创建 `Assets/Scripts/UI/settingPanel.cs` - 设置面板脚本
- 创建 `Assets/Scripts/UI/GetGoodsPanel.cs` - 获取物品面板脚本

### 5. 完善UI配置
- 确保 `UIPanelType.json` 配置正确
- 创建 `UIPanelType` 枚举类
- 完善 `basePanel.cs` 基类

### 6. 添加UI资源
- 添加按钮、面板背景等UI素材
- 添加必要的字体和图标
- 确保所有UI元素具有合适的样式

## 技术要点
- 使用Unity UGUI系统创建UI
- 遵循现有的UI管理框架
- 确保UI在不同分辨率下正常显示
- 添加适当的动画和过渡效果
- 确保与现有登录系统的兼容性

## 预期成果
- 完整的主页面UI设计
- 所有必要的面板预制体
- 功能完整的面板脚本
- 与现有系统的无缝集成

此计划将为游戏提供一个美观、功能完整的主页面UI，符合3D皇室战争类游戏的风格和需求。
