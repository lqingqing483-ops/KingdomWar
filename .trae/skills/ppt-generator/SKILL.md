# PPT Generator

This skill generates professional PowerPoint presentations based on provided technical documentation or outlines.

## When to Invoke

Invoke this skill when:
- User requests to create a PowerPoint presentation
- User wants to convert technical documentation into a presentation
- User needs a professional slideshow for project introduction
- User asks for visual representation of technical content

## How to Use

1. **Provide Context**: Include the technical documentation or outline that the PPT should be based on
2. **Specify Requirements**: Mention any specific requirements such as length, style, or emphasis areas
3. **Request Generation**: Ask the skill to generate the PPT

## Output Format

The skill will generate a PowerPoint presentation with:
- Professional design and layout
- Clear structure and organization
- Relevant visuals and diagrams
- Code snippets (if applicable)
- Consistent styling throughout

## Example Usage

"Generate a PPT based on the KingdomWar technical documentation, focusing on the core features and technical architecture."

"Create a presentation that highlights the performance optimization techniques used in the project."

## Best Practices

- Provide detailed technical documentation for the most accurate results
- Specify the target audience (e.g., teachers, interviewers, team members)
- Mention any specific sections that should be emphasized
- Indicate the desired length or time limit for the presentation
