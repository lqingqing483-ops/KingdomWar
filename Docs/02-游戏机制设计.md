# 游戏机制设计

## 1. 核心玩法机制

### 1.1 基础规则

#### 1.1.1 战场布局
- **战场尺寸**: 40x40 单位（Unity世界坐标）
- **玩家区域**: 每方拥有20x40的半场
- **河道**: 中间4单位宽的河道
- **防御塔位置**: 
  - 主塔：位于后方中心
  - 副塔：位于前方两侧
- **桥**: 连接两岸的桥梁（每方2座）

#### 1.1.2 胜利条件
- **主要胜利**: 摧毁敌方主塔
- **次要胜利**: 3分钟时间结束时，摧毁更多塔的一方获胜
- **平局**: 双方摧毁相同数量的塔，进入1分钟加时赛
- **加时赛规则**: 加时赛中首先摧毁任意塔的一方获胜

#### 1.1.3 时间限制
- **常规时间**: 3分钟（180秒）
- **加时赛**: 1分钟（60秒）
- **突然死亡**: 加时赛结束后如果仍平局，进入突然死亡模式，先得分者获胜

### 1.2 资源系统

#### 1.2.1 圣水（Elixir）
- **初始圣水**: 5点
- **圣水恢复速度**: 每秒恢复1点（双倍圣水时间每秒2点）
- **圣水上限**: 10点
- **双倍圣水时间**: 最后60秒自动触发

#### 1.2.2 圣水消耗
每张卡牌都有固定的圣水消耗，范围从1-10点不等：
- **低消耗（1-3点）**: 基础单位、小型法术
- **中消耗（4-6点）**: 中型单位、中型法术
- **高消耗（7-10点）**: 大型单位、终极法术

#### 1.2.3 圣水策略
- **圣水优势**: 保持比对手更多的圣水储备
- **圣水陷阱**: 快速消耗圣水引诱对手反击
- **圣水管理**: 合理分配圣水用于进攻和防守

## 2. 卡牌系统

### 2.1 卡牌分类

#### 2.1.1 单位卡牌
**近战单位**:
- 骑士（3圣水）
- 巨人（5圣水）
- 野蛮人（5圣水）
- 皮卡超人（7圣水）
- 王子（5圣水）

**远程单位**:
- 弓箭手（3圣水）
- 火枪手（4圣水）
- 迫击炮（4圣水）
- 女巫（5圣水）
- 三个火枪手（9圣水）

**飞行单位**:
- 亡灵（3圣水）
- 亡灵大军（5圣水）
- 飞龙宝宝（4圣水）
- 熔岩猎犬（7圣水）

**建筑单位**:
- 炮塔（4圣水）
- 迫击炮（4圣水）
- X连弩（6圣水）
- 电磁塔（4圣水）

#### 2.1.2 法术卡牌
**攻击法术**:
- 火球（4圣水）
- 箭雨（3圣水）
- 闪电（6圣水）
- 火箭（6圣水）
- 毒药法术（4圣水）

**防御法术**:
- 冰冻法术（4圣水）
- 电击法术（2圣水）
- 狂暴法术（2圣水）
- 复仇（3圣水）

**增益法术**:
- 治疗（1圣水）
- 镜像（1圣水）
- 克隆（3圣水）

### 2.2 卡牌属性

#### 2.2.1 基础属性
每个单位卡牌包含以下属性：
- **生命值**: 单位的总生命值
- **攻击力**: 每次攻击造成的伤害
- **攻击速度**: 攻击间隔（秒）
- **移动速度**: 单位移动速度
- **射程**: 攻击范围
- **部署时间**: 从部署到可以行动的时间
- **目标类型**: 地面/空中/建筑

#### 2.2.2 特殊属性
- **冲锋**: 首次攻击造成双倍伤害
- **分裂**: 死亡时分裂成多个小单位
- **溅射伤害**: 攻击对范围内所有敌人造成伤害
- **治疗**: 攻击时治疗友方单位
- **嘲讽**: 强制敌方单位攻击自己

### 2.3 卡牌升级

#### 2.3.1 升级机制
- **升级材料**: 收集相同卡牌的碎片
- **升级等级**: 从1级到13级
- **升级奖励**: 
  - 生命值提升10%
  - 攻击力提升10%
  - 特殊效果增强

#### 2.3.2 升级需求
| 等级 | 需要碎片 | 金币 |
|------|----------|------|
| 1-2 | 2 | 100 |
| 2-3 | 4 | 200 |
| 3-4 | 10 | 400 |
| 4-5 | 20 | 800 |
| 5-6 | 50 | 1600 |
| 6-7 | 100 | 3200 |
| 7-8 | 200 | 6400 |
| 8-9 | 400 | 12800 |
| 9-10 | 800 | 25600 |
| 10-11 | 1000 | 51200 |
| 11-12 | 2000 | 102400 |
| 12-13 | 4000 | 204800 |

### 2.4 卡组系统

#### 2.4.1 卡组构成
- **卡组大小**: 8张卡牌
- **卡组槽位**: 玩家可以保存多个卡组
- **卡组限制**: 同一张卡牌在一个卡组中只能出现一次（镜像卡牌除外）

#### 2.4.2 卡组类型
- **进攻型**: 以高伤害单位为主，快速推进
- **防守型**: 以防守单位和建筑为主，反击取胜
- **平衡型**: 攻守兼备，适应性强
- **消耗型**: 使用法术和远程单位消耗对手

#### 2.4.3 卡组构建原则
1. **圣水曲线**: 确保卡组平均圣水消耗在3.5-4.5之间
2. **功能覆盖**: 包含进攻、防守、法术等不同功能
3. **协同效应**: 卡牌之间要有良好的配合
4. **应对能力**: 能够应对多种类型的对手

## 3. 战斗系统

### 3.1 单位行为

#### 3.1.1 寻路系统
- **A*算法**: 用于单位寻路
- **碰撞检测**: 单位之间有碰撞体积
- **路径优化**: 自动寻找最短路径
- **动态避障**: 避开障碍物和其他单位

#### 3.1.2 攻击逻辑
- **目标选择**: 
  1. 优先攻击最近的敌方单位
  2. 如果有多个目标距离相同，优先攻击建筑
  3. 某些单位有特殊目标偏好
- **攻击范围**: 只有在射程内才能攻击
- **攻击动画**: 攻击前有准备动画
- **攻击判定**: 攻击动画结束时造成伤害

#### 3.1.3 单位AI
- **基础AI**: 自动寻路和攻击
- **高级AI**: 
  - 协同作战：同类型单位会一起行动
  - 战术调整：根据战场情况调整策略
  - 优先级判断：优先攻击高威胁目标

### 3.2 建筑系统

#### 3.2.1 建筑类型
**防御塔**:
- 主塔：高生命值，高伤害
- 副塔：中等生命值，中等伤害
- 炮塔：远程攻击，低生命值

**功能建筑**:
- 迫击炮：远程范围攻击
- 电磁塔：快速攻击，短射程
- X连弩：超远程攻击

**资源建筑**:
- 圣水收集器：缓慢产生圣水
- 金矿：产生金币（如果实现）

#### 3.2.2 建筑属性
- **生命值**: 建筑的总生命值
- **攻击力**: 每次攻击造成的伤害
- **攻击速度**: 攻击间隔
- **射程**: 攻击范围
- **持续时间**: 建筑存在时间（部分建筑）
- **占地**: 建筑占用的空间

#### 3.2.3 建筑机制
- **部署限制**: 只能在己方半场部署
- **建筑上限**: 同时存在的建筑数量有限制
- **建筑攻击**: 建筑可以攻击范围内的敌方单位
- **建筑摧毁**: 建筑被摧毁时可能触发特殊效果

### 3.3 法术系统

#### 3.3.1 法术类型
**单体法术**:
- 电击法术：对单个目标造成伤害并眩晕
- 治疗法术：治疗单个友方单位

**范围法术**:
- 火球：对圆形区域造成伤害
- 箭雨：对矩形区域造成伤害
- 毒药法术：对区域持续造成伤害

**特殊法术**:
- 冰冻法术：冻结区域内的单位
- 狂暴法术：提升区域内单位的攻击速度和移动速度
- 镜像：复制最后使用的卡牌

#### 3.3.2 法术效果
- **伤害**: 直接减少目标生命值
- **控制**: 眩晕、冻结、减速
- **增益**: 提升单位属性
- **治疗**: 恢复单位生命值
- **特殊**: 镜像、克隆等独特效果

#### 3.3.3 法术机制
- **即时生效**: 部署后立即产生效果
- **持续时间**: 某些法术有持续时间
- **叠加规则**: 多个法术效果是否可以叠加
- **抗性**: 某些单位对特定法术有抗性

## 4. 经济系统

### 4.1 货币系统

#### 4.1.1 金币（Gold）
- **获取方式**:
  - 开启宝箱
  - 完成任务
  - 捐赠卡牌
  - 参与活动
- **用途**:
  - 升级卡牌
  - 购买卡牌
  - 购买宝箱

#### 4.1.2 宝石（Gems）
- **获取方式**:
  - 免费宝箱
  - 任务奖励
  - 活动奖励
  - 内购
- **用途**:
  - 购买金币
  - 加速宝箱开启
  - 购买特殊宝箱
  - 购买皮肤

#### 4.1.3 卡牌碎片（Card Fragments）
- **获取方式**:
  - 开启宝箱
  - 捐赠卡牌
  - 任务奖励
- **用途**:
  - 升级卡牌
  - 捐赠给公会成员

### 4.2 宝箱系统

#### 4.2.1 宝箱类型
- **免费宝箱**: 每4小时获得一个
- **皇冠宝箱**: 每天收集10个皇冠获得
- **胜利宝箱**: 赢得对战后获得
- **特殊宝箱**: 活动或任务获得
- **传奇宝箱**: 极低概率获得

#### 4.2.2 宝箱内容
- **卡牌**: 随机卡牌和碎片
- **金币**: 固定数量金币
- **宝石**: 少量宝石（特殊宝箱）
- **特殊物品**: 皮肤、表情等

#### 4.2.3 宝箱开启
- **宝箱槽位**: 玩家有4个宝箱槽位
- **开启时间**: 不同宝箱有不同的开启时间
- **加速开启**: 使用宝石加速开启
- **立即开启**: 使用更多宝石立即开启

### 4.3 商店系统

#### 4.3.1 每日商店
- **刷新时间**: 每天刷新一次
- **商品内容**:
  - 3张随机卡牌
  - 1张传奇卡牌（每周一次）
  - 特殊宝箱
- **购买限制**: 每种商品只能购买一次

#### 4.3.2 特殊商店
- **活动商店**: 限时活动期间开放
- **赛季商店**: 赛季期间开放
- **公会商店**: 使用公会货币购买

## 7. 平衡性设计

### 7.1 卡牌平衡

#### 7.1.1 平衡原则
- **多样性**: 确保没有绝对最强的卡组
- **可应对性**: 每种策略都有应对方法
- **技能平衡**: 技术比卡牌更重要
- **动态调整**: 根据数据持续调整

#### 7.1.2 平衡方法
- **数据监控**: 收集卡牌使用率和胜率数据
- **玩家反馈**: 收集玩家反馈意见
- **定期调整**: 每月进行一次平衡调整
- **小幅调整**: 每次调整幅度不超过10%

### 7.2 匹配系统

#### 7.2.1 匹配规则
- **杯数匹配**: 优先匹配杯数相近的对手
- **卡牌等级**: 考虑卡牌等级差异
- **胜率平衡**: 平衡双方胜率
- **网络延迟**: 确保网络延迟可接受

#### 7.2.2 匹配算法
- **ELO系统**: 使用ELO评分系统
- **匹配池**: 维护一个匹配池
- **超时机制**: 匹配超时后放宽条件
- **防连胜**: 避免连续匹配到相同对手

## 8. 总结

KingdomWar的游戏机制设计融合了皇室战争的核心玩法，并通过3D视角和增强的策略深度提供了独特的游戏体验。卡牌系统、战斗系统、经济系统和竞技场系统相互配合，形成了完整且富有深度的游戏循环。

**核心优势**:
1. **策略深度**: 多样化的卡牌组合和战术选择
2. **平衡性**: 持续的平衡调整确保公平性
3. **社交性**: 好友和公会系统增强社交互动
4. **可扩展性**: 模块化设计便于添加新内容

通过精心设计的游戏机制，KingdomWar将为玩家提供长期、有趣且富有挑战性的游戏体验。
