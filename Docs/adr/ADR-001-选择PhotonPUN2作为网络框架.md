# ADR-001: 选择 Photon PUN2 作为网络框架

**日期**: 2026-07-05  
**状态**: 已实施  
**影响范围**: 网络层、匹配、帧同步

## 背景
需要为实时对战选择网络框架。

## 选项
1. **Photon PUN2** — 当前使用
2. **Mirror Networking** — 文档曾推荐
3. **Netcode for GameObjects** — Unity 官方
4. **自制 TCP/UDP** — 自定义实现

## 决策
选择 Photon PUN2，理由：
- 已集成并运行稳定
- 有成熟的房间匹配和 RPC 机制
- 免费层足够小规模测试
- 社区资源丰富

## 后果
- ✅ 帧同步 + 状态同步均可用
- ✅ 支持 2 人对战
- ⚠️ 免费层有限制并发
- ⚠️ 替换为自建服务器需要重构网络层
