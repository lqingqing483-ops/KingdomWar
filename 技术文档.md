# KingdomWar 项目技术文档

## 1. 项目背景与目标概述

### 1.1 项目背景
KingdomWar是一款基于皇室战争玩法的3D实时策略对战游戏，旨在通过创新的3D视角和增强的游戏机制，为玩家提供全新的策略对战体验。在一个充满魔法与战争的中世纪幻想世界中，各个王国为了争夺资源和领土而不断战斗。玩家作为王国的指挥官，通过召唤各种单位和施放法术来摧毁敌方城堡。

### 1.3 技术挑战
- **3D渲染性能**：确保在各种硬件配置下保持流畅的游戏体验
- **网络同步**：实现低延迟的实时对战体验
- **AI行为**：构建智能的单位AI系统
- **资源管理**：优化内存使用和资源加载

```csharp
public class ResourceManager : MonoBehaviour
{
    public static ResourceManager Instance { get; private set; }
    
    private Dictionary<string, Object> loadedResources = new Dictionary<string, Object>();
    
    void Awake()
    {
        Instance = this;
    }
    
    public void LoadResourceAsync<T>(string path, Action<T> callback) where T : Object
    {
        if (loadedResources.ContainsKey(path))
        {
            callback?.Invoke(loadedResources[path] as T);
            return;
        }
        
        StartCoroutine(LoadResourceCoroutine(path, callback));
    }
    
    private IEnumerator LoadResourceCoroutine<T>(string path, Action<T> callback) where T : Object
    {
        ResourceRequest request = Resources.LoadAsync<T>(path);
        
        yield return request;
        
        if (request.asset != null)
        {
            loadedResources[path] = request.asset;
            callback?.Invoke(request.asset as T);
        }
    }
}
```

### 4.4 对象池系统
**功能说明**：管理游戏对象的创建和回收，减少内存分配和GC压力。

**实现细节**：
- 预创建对象，避免运行时频繁实例化
- 对象回收和重用机制
- 支持不同类型对象的池管理

**关键代码**：
```csharp
public class ObjectPool : MonoBehaviour
{
    public static ObjectPool Instance { get; private set; }
    
    [System.Serializable]
    public class Pool
    {
        public string tag;
        public GameObject prefab;
        public int size;
    }
    
    public List<Pool> pools;
    private Dictionary<string, Queue<GameObject>> poolDictionary;
    
    void Awake()
    {
        Instance = this;
    }
    
    void Start()
    {
        poolDictionary = new Dictionary<string, Queue<GameObject>>();
        
        foreach (Pool pool in pools)
        {
            Queue<GameObject> objectPool = new Queue<GameObject>();
            
            for (int i = 0; i < pool.size; i++)
            {
                GameObject obj = Instantiate(pool.prefab);
                obj.SetActive(false);
                objectPool.Enqueue(obj);
            }
            
            poolDictionary.Add(pool.tag, objectPool);
        }
    }
    
    public GameObject SpawnFromPool(string tag, Vector3 position, Quaternion rotation)
    {
        if (!poolDictionary.ContainsKey(tag))
        {
            Debug.LogWarning($"Pool with tag {tag} doesn't exist");
            return null;
        }
        
        GameObject objectToSpawn = poolDictionary[tag].Dequeue();
        
        objectToSpawn.SetActive(true);
        objectToSpawn.transform.position = position;
        objectToSpawn.transform.rotation = rotation;
        
        IPooledObject pooledObj = objectToSpawn.GetComponent<IPooledObject>();
        if (pooledObj != null)
        {
            pooledObj.OnObjectSpawn();
        }
        
        poolDictionary[tag].Enqueue(objectToSpawn);
        
        return objectToSpawn;
    }
}
```

## 5. 关键技术点与难点攻克方案

### 5.1 3D渲染优化
**技术点**：3D场景渲染性能优化

**难点**：确保在不同硬件配置下保持流畅的帧率

**解决方案**：
- **网格合并**：减少Draw Call
- **纹理图集**：减少材质数量和Draw Call
- **光照优化**：使用烘焙光照，减少实时光源数量

**关键代码**：
```csharp
public class LODController : MonoBehaviour
{
    [System.Serializable]
    public class LODLevel
    {
        public float distance;
        public GameObject lodObject;
    }
    
    [SerializeField] private LODLevel[] lodLevels;
    private LODLevel currentLOD;
    
    void Update()
    {
        float distance = Vector3.Distance(Camera.main.transform.position, transform.position);
        LODLevel bestLOD = null;
        
        foreach (var lod in lodLevels)
        {
            if (distance <= lod.distance)
            {
                if (bestLOD == null || lod.distance < bestLOD.distance)
                {
                    bestLOD = lod;
                }
            }
        }
        
        if (bestLOD != currentLOD)
        {
            SwitchLOD(bestLOD);
        }
    }
}
```

**难点**：网络延迟和状态同步

**解决方案**：

```csharp
public class DeltaSync : MonoBehaviour
{
    private NetworkState lastState;
    private float syncInterval = 0.1f;
    private float lastSyncTime;
    
    void Update()
    {
        if (Time.time - lastSyncTime < syncInterval)
            return;
        
        lastSyncTime = Time.time;
        SendDeltaSync();
    }
    
    private void SendDeltaSync()
    {
        NetworkState currentState = GetCurrentState();
        NetworkDelta delta = CalculateDelta(lastState, currentState);
        
        if (delta.HasChanges())
        {
            SendDelta(delta);
            lastState = currentState;
        }
    }
}
```

**状态接口定义**：

```csharp
public interface IState
{
    string StateName { get; }
    void OnEnter(StateMachine fsm);
    void OnUpdate(float deltaTime);
    void OnExit(StateMachine fsm);
    bool CanTransitionTo(string nextStateName);
}

public abstract class BaseState : IState
{
    public string StateName { get; protected set; }
    protected StateMachine fsm;
    
    public virtual void OnEnter(StateMachine fsm)
    {
        this.fsm = fsm;
    }
    
    public virtual void OnUpdate(float deltaTime) { }
    
    public virtual void OnExit(StateMachine fsm) { }
    
    public virtual bool CanTransitionTo(string nextStateName)
    {
        return true;
    }
}
```

**状态机核心**：
```csharp
public class StateMachine : MonoBehaviour
{
    [SerializeField] private string currentStateName;
    
    private Dictionary<string, IState> states;
    private IState currentState;
    private Queue<FSMEvent> eventQueue;
    private List<Transition> transitions;
    
    public string CurrentState => currentStateName;
    
    public void Initialize()
    {
        states = new Dictionary<string, IState>();
        eventQueue = new Queue<FSMEvent>();
        transitions = new List<Transition>();
    }
    
    public void AddState(IState state)
    {
        if (!states.ContainsKey(state.StateName))
        {
            states[state.StateName] = state;
        }
    }
    
    public void AddTransition(string fromState, string toState, FSMEventType eventType)
    {
        transitions.Add(new Transition(fromState, toState, eventType));
    }
    
    public void ChangeState(string newStateName)
    {
        if (!states.ContainsKey(newStateName)) return;
        
        IState newState = states[newStateName];
        
        if (currentState != null && !currentState.CanTransitionTo(newStateName)) return;
        
        currentState?.OnExit(this);
        currentState = newState;
        currentStateName = newStateName;
        currentState.OnEnter(this);
    }
    
    public void SendEvent(FSMEvent fsmEvent)
    {
        eventQueue.Enqueue(fsmEvent);
    }
    
    public void Update()
    {
        ProcessEvents();
        currentState?.OnUpdate(Time.deltaTime);
    }
    
    private void ProcessEvents()
    {
        while (eventQueue.Count > 0)
        {
            FSMEvent fsmEvent = eventQueue.Dequeue();
            ProcessTransition(fsmEvent);
        }
    }
    
    private void ProcessTransition(FSMEvent fsmEvent)
    {
        foreach (var transition in transitions)
        {
            if (transition.fromState == currentStateName && 
                transition.eventType == fsmEvent.Type)
            {
                ChangeState(transition.toState);
                break;
            }
        }
    }
}
```

```csharp
        this.unit = unit;

    }

    {
        base.OnEnter(fsm);
        unit.PlayAnimation("Idle");
    }
    
    public override void OnUpdate(float deltaTime)
    {
        if (unit.FindTarget() != null)
        {
            fsm.SendEvent(new FSMEvent(FSMEventType.TargetFound));
        }
    }
    
    public override bool CanTransitionTo(string nextStateName)
    {
        return nextStateName == "Moving" || 
               nextStateName == "Attacking" || 
               nextStateName == "Dead";
    }
}

public class AttackingState : BaseState
{
    private Unit unit;
    private float attackTimer;
    
    public AttackingState(Unit unit)
    {
        this.unit = unit;
        StateName = "Attacking";
    }
    
    public override void OnEnter(StateMachine fsm)
    {
        base.OnEnter(fsm);
        attackTimer = 0f;
    }
    
    public override void OnUpdate(float deltaTime)
    {
        if (unit.Target == null || unit.Target.IsDead)
        {
            fsm.SendEvent(new FSMEvent(FSMEventType.TargetLost));
            return;
        }
        
        attackTimer += deltaTime;
        if (attackTimer >= unit.Stats.attackSpeed)
        {
            unit.PerformAttack();
            attackTimer = 0f;
        }
    }
}
```

```csharp
public class MemoryMonitor : MonoBehaviour
{
    [SerializeField] private float memoryCheckInterval = 5f;
    private float lastCheckTime;
    private long totalMemory;
    
    void Update()
    {
        if (Time.time - lastCheckTime < memoryCheckInterval)
            return;
        
        lastCheckTime = Time.time;
        CheckMemoryUsage();
    }
    
    private void CheckMemoryUsage()
    {
        totalMemory = GC.GetTotalMemory(false);
        
        Debug.Log($"Total Memory: {totalMemory / 1024 / 1024} MB");
        
        if (totalMemory > 2000000000) // 2GB
        {
            Debug.LogWarning("High memory usage detected!");
            TriggerGarbageCollection();
        }
    }
    
    private void TriggerGarbageCollection()
    {
        GC.Collect();
        GC.WaitForPendingFinalizers();
        Resources.UnloadUnusedAssets();
    }
}
```
